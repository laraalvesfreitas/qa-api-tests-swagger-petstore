## Caso de teste CT-001

**Título:** Criar pet com dados mínimos válidos

**Descrição:** Validar que a API permite cadastrar um novo pet com informações corretas.

**Endpoint:**  `/pet`

**Método:** POST

**Pré-condição:**

- API disponível

**Request Body:**

``
json
{
  "id": 10,
  "name": "Bob",
  "status": "available"
}
``

**Passos:** 
1. Enviar requisição POST para `/pet` com dados do válidos do pet.  

**Resultado Esperado:**

- Status code **200 OK**
- Pet criado com sucesso
- Dados retornados iguais ao request
- A resposta pode conter campos adicionais definidos pelo modelo da API, como `photoUrls` e `tags`, mesmo que não enviados na requisição.
