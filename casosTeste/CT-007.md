## Caso de teste CT-007

**Título:** Buscar pet por ID inválido

**Descrição:** Validar que a API não permite buscar um pet utilizando um ID inválido (mal formatado), contendo espaços ou caracteres não numéricos.

**Endpoint:**  `/pet/{petId}`

**Método:** GET

**Pré-condição:**

- API disponível
- Utilizar um ID em formato inválido (com espaços ou caracteres não numéricos)

**Passos:** 
1. Enviar requisição GET para `/pet/{petId}` com um ID inválido (ex: `55 ` ou `55\n`)
**Resultado Esperado:**

- Status code **400 Bad Request**
- Mensagem indicando formato inválido do ID
- Resposta amigável sem exposição de erro interno
