## Caso de teste CT-009

**Título:** Excluir pet por ID existente

**Descrição:** Validar que a API permite excluir um pet existente a partir do ID informado.

**Endpoint:**  `/pet/{petId}`

**Método:** DELETE

**Pré-condição:**

- API disponível
- Pet previamente cadastrado com o ID informado


**Passos:** 
1. Enviar requisição DELETE para `/pet/{petId}` com um ID existente (ex: 12)
   
**Resultado Esperado:**

- Status code **200 OK**
- Pet excluído com sucesso
- Resposta indica que a operação foi realizada
- Após exclusão, ao buscar o mesmo ID, a API deve retornar **404 Not Found**
