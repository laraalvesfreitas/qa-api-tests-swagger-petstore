## Caso de teste CT-003

**Título:** Atualizar pet com dados válidos  

**Descrição:** Validar que a API permite atualizar os dados de um pet existente

**Endpoint:**  `/pet`

**Método:** PUT

**Pré-condição:**

- API disponível

**Request Body:**

````json
{
  "id": 31,
  "name": "Joaquim"
}
````

**Passos:** 
1. 1. Enviar requisição PUT para `/pet` com dados atualizados do pet

**Resultado Esperado:**

- Status code **200 OK**
- Pet atualizado com sucesso
- Dados retornados refletem a atualização realizada
- Campo "name" atualizado para "Joaquim"
- Dados retornados refletem a atualização realizada
-Estrutura da resposta conforme modelo da API
-Campos retornados com tipos corretos
