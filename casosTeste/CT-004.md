## Caso de teste CT-004

**Título:** Atualizar pet com dados inválidos ou nulos  

**Descrição:** Validar que a API não permite atualizar os dados de um animal caso contenha informações nulas ou incorretas.

**Endpoint:**  `/pet`

**Método:** PUT

**Pré-condição:**

- API disponível

**Request Body:**

````json
{
  "id": "QBER",
  "name": "Joaquim"
}
````

**Passos:** 

1. Enviar requisição PUT para `/pet` com dados inválidos. 

**Resultado Esperado:**

- Status code **400**
- Mensagem de erro retornada na resposta
- Mensagem indicando tipo inválido para o campo id
- Estrutura da resposta conforme modelo da API
