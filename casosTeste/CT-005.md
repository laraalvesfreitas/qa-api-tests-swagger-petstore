## Caso de teste CT-005

**Título:** Fazer upload de imagem do pet. 

**Descrição:** Validar que a API permite realizar o upload de imagem associada a um pet existente.

**Endpoint:**  `/pet/{petId}/uploadImage`

**Método:** POST

**Pré-condição:**

- API disponível
- Existir um pet cadastrado com o ID informado
  

**Request Body:**

````form-data
- file: arquivo de imagem (tipo File)
- additionalMetadata: texto descritivo da imagem (opcional)

````

**Passos:** 

1. Informar ID do pet na url ({petId})
2. Enviar no form-data um arquivo (File) e um texto opcional falando sobre o que é a imagem (additionalMetadata)

**Resultado Esperado:**
- Status code **200 OK**
- Upload realizado com sucesso
- Resposta:
  - Nome do arquivo enviado
  - Conteúdo de `additionalMetadata`
