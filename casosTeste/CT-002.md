## Caso de teste CT-002

**Título:** Criar pet com todos os campos preenchidos

**Descrição:** Validar que a API permite cadastrar um novo pet com todos as informações preenchidas.

**Endpoint:**  `/pet`

**Método:** POST

**Pré-condição:**

- API disponível

**Request Body:**

````json
{
  "id": 31,
  "category": {
    "id": 2,
    "name": "Cachorros"
  },
  "name": "Meg",
  "photoUrls": [
    "https://images.dog.ceo/breeds/appenzeller/n02107908_6683.jpg"
  ],
  "tags": [
    {
      "id": 9,
      "name": "pets"
    }
  ],
  "status": "available"
}
````

**Passos:** 
1. Enviar requisição POST para `/pet` com todos os dados preenchidos.  

**Resultado Esperado:**

- Status code **200 OK**
- Pet criado com sucesso
- Dados retornados correspondem ao request enviado:
  - id
  - category
  - name
  - photoUrls
  - tags
  - status
- Estrutura da resposta conforme modelo da API
- Campos retornados com tipos corretos (string, number, array)
