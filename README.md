#  Testes de API - Swagger Petstore

## 📌 Sobre o projeto

Este repositório apresenta um projeto de testes de API realizado na Swagger Petstore, com foco na validação da funcionalidade **Pet**.

Foram executados testes funcionais abrangendo o fluxo completo de CRUD (Create, Read, Update, Delete), incluindo cenários positivos e negativos, com validação de status codes, estrutura de resposta e regras de negócio da API.

O projeto contempla a criação e execução de casos de teste, documentação de evidências e registro de possíveis falhas, seguindo boas práticas de Quality Assurance.

As requisições foram realizadas utilizando o Postman, com organização dos testes e evidências para garantir rastreabilidade e reprodutibilidade.

Este projeto tem como objetivo demonstrar habilidades práticas em testes de API, pensamento analítico e capacidade de identificação de inconsistências em serviços REST.


Obs: A API retorna campos adicionais com valores padrão mesmo quando não enviados no request, indicando um comportamento padrão do modelo de dados.
