## Bug Report BUG-001

**Título:** API permite criação de pet sem campos obrigatórios.

**Endpoint:** `/pet`  
**Método:** POST  


**Ambiente:** Swagger Petstore (Postman)

**Descrição:**  A API permite a criação de um pet sem os campos obrigatórios `name` e `photoUrls`, mesmo estando marcados como obrigatórios na documentação.


**Pré-condição:**  
- API disponível

**Passos para reproduzir:**  
1. Enviar requisição POST para `/pet` com o seguinte body:
`` Json
{
  "id": 11
}
``

**Resultado Esperado:**  

- A API deve retornar erro (status 400(Bad Request))
- Deve informar que os campos name e photoUrls são obrigatórios

**Resultado Atual:**  

- A API retorna status 200
- O pet é criado com sucesso mesmo sem os campos obrigatórios

**Severidade:** Média

**Prioridade:** Média
