## Bug Report BUG-002

**Título:** API retorna erro 500 ao receber tipo inválido no campo id

**Endpoint:** `/pet`  
**Método:** PUT  

**Ambiente:** Swagger Petstore (Postman)

**Descrição:**  
Ao enviar um valor inválido (string) no campo `id`, a API retorna erro 500 (Internal Server Error), indicando falha interna, ao invés de validar a entrada e retornar erro adequado (400 Bad Request).

**Pré-condição:**  
- API disponível

**Passos para reproduzir:**  
1. Enviar requisição PUT para `/pet` com o seguinte body:
```json
{
  "id": "QBER",
  "name": "Joaquim"
}
````

**Resultado Esperado:**  

- Status code **400 Bad Request**
- Mensagem de erro indicando tipo inválido para o campo `id`

**Resultado Atual:**  

- Status code **500 Internal Server Error**
- A API não trata o erro de entrada corretament


**Severidade:** Alta

**Prioridade:** Alta
