## Bug Report BUG-003

**Título:** API expõe erro interno ao buscar pet com ID inválido

**Endpoint:** `/pet/{petId}`  
**Método:** GET  


**Ambiente:** Swagger Petstore (Postman)

**Descrição:**  A API retorna mensagem contendo detalhes internos da aplicação (java.lang.NumberFormatException) ao receber um ID inválido com caracteres não numéricos ou espaços, ao invés de tratar o erro de forma adequada.


**Pré-condição:**  
- API disponível
- Utilizar um ID em formato inválido (com espaços ou caracteres não numéricos)


**Passos para reproduzir:**  
1. Enviar requisição GET para `/pet/{petId}` com um ID inválido (ex: `55 ` ou `55\n`)  


**Resultado Esperado:**  

- Status code **400 Bad Request**
- Mensagem indicando formato inválido do ID
- Resposta amigável sem exposição de erro interno  

**Resultado Atual:**  

- A API retorna status 404
- A resposta expõe detalhes internos da aplicação:
  `java.lang.NumberFormatException`

**Severidade:** Média

**Prioridade:** Média
